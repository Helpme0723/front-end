## 📁 **폴더 구조 및 환경 변수**

<details>

**<summary>Directory Structure</summary>**

```

📦src
 ┣ 📂apis // API 연결
 ┣ 📂assets
 ┣ 📂components // 재사용 가능한 UI 컴포넌트
 ┣ 📂context // 애플리케이션 상태 관리
 ┣ 📂layouts // 레이아웃 컴포넌트
 ┣ 📂pages // 페이지 구성 컴포넌트
 ┣ 📂styles // CSS
 ┣ App.css
 ┣ App.jsx
 ┣ App.test.js
 ┣ index.css
 ┣ index.jsx
 ┣ logo.svg // 파비콘
 ┣ reportWebVitals.js
 ┗ setupTests.js

```

</details>

## 시작 가이드

**installation**

```
$ git clone https://github.com/Helpme0723/front-end.git
$ cd front-end
```

**copy**

```
.env
```

**BackEnd**

```
$ npm ci
$ npm start
```
